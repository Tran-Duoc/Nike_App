<hr>

### Để sử dụng project thì

-  Clone project bằng lệnh sao

```shell
   git clone https://github.com/Tran-Duoc/Nike_App.git
```

-  Sao khi clone xong

```shell
   cd src
   npm install
   npm run dev
```
